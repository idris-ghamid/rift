# Rift Example App

A comprehensive Flutter demo app showcasing all 75+ features of the Rift NoSQL database.

## Features

- **75+ Interactive Demos** — Each feature has a working demo with description, code example, and live output
- **7 Feature Categories** — Query & Search, Data & Schema, Security & Privacy, Performance & Storage, Sync & Distribution, Developer Tools, Platform & Integration
- **Apple-Style UI** — Clean, minimal, professional design with dark/light mode
- **Search & Filter** — Find any feature instantly
- **Auto-Run** — Demos execute automatically when opened

## Getting Started

```bash
cd hive/example
flutter pub get
flutter run
```

## Architecture

- `main.dart` — App entry point with Rift initialization and theming
- `home_page.dart` — Feature listing with category pills and search
- `about_page.dart` — About the creator
- `widgets/demo_page.dart` — Reusable demo scaffold with About/Code/Output sections
- `demos/` — 75+ individual demo files
